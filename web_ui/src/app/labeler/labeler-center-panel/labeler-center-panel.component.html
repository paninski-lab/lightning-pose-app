Use camera parameters to assist labeling.

With accurate parameters, you need only label two views per keypoint: the other views can be auto-labeled by triangulating and reprojecting.

How to provide camera parameters to Lightning Pose

Camera calibration is typically performed using Anipose before every session. Camera parameters are recommended to be session-specific to allow for changes in camera position from session to session. A project-wide calibration file can be used as a fallback.
